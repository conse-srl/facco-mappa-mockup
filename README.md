## Installation

Clone the project

```bash
  git clone https://github.com/kevinbalboni/openui5-calendar.git
```
Go to the project directory

```bash
  cd openui5-calendar
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
